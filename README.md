# EduTrack – Student Management System

A full stack web application to manage student records such as adding, viewing, updating, and deleting student information.

## Tech Stack

* Python
* Django
* MySQL
* HTML, CSS, Bootstrap
* JavaScript
* React

## Features

* Add Student
* View Students
* Update Student Details
* Delete Students
* Search Students
* Dashboard for student data overview
